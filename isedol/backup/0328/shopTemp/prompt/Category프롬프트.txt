카테고리 페이지 UI 제작

설명 : 

해당 프로젝트는 spa방식으로 제작될 것이며 pc와 모바일 디바이스의 UX를 고려 제작해라

이번에는 category 페이지를 재작할 것이다
데이터는 더미데이터를 생성해서 작성하라. 더미데이터는 이름과 가격, 메이커, 분류(상의, 하의, 아우터, 악세서리), 성별, 색상 정보로 존재한다.

필요한 라이브러리는 cdn방식으로 선언하되 현재 작성되어있는 내용에서 사용하는 것을 권장한다
스타일에서 중복되는 내용은 변수로 선언해 사용해라

최 상단은 home페이지에서 제작한 nav를 동일하게 가져온다 즉 사이드바인 user사이드바도 동일하게 가져와 동일하게 작동해야한다

Category페이지는 다음과 같이 구성된다
전체 콘텐츠 영역 width = 1200px
좌측 사이드바
- 다음은 좌측 외 부분에 출력되는 데이터의 분류를 정의한다
분류 카테고리는 다음과 같다
1. 성별 / 2. 메이커 / 3. 분류 / 4. 색상 / 5. 가격


좌측 외 상단 = swiper로 구성된 슬라이드 
좌측 외 부분의 상단 외 = grid를 사용한 상품 출력 (더미데이터로 존재하기 때문에 단색 카드로 존재하며 이름과 가격은 더미데이터를 따른다)
출력할 때 사이드바에서 정의된 분류에 따라 구분되며 분류된 내용 출력시 상단에 분류 기준 및 분류된 데이터의 이름을 명시하라
예시 ) 데이터의 메이커 - 프라다
메이커 (프라다)
데이터 데이터 데이터 => (grid)로 출력
grid로 데이터를 출력하는 경우 화면에서 넘어간다면 해당 영역에만 스크롤을 적용하여 nav, 좌측사이드바, footer는 움직이지 않게 만들어라

하단에는 home에서 만든 section6 즉 footer를 넣어라


위 사항 및 내용을 준수해 기존 코드에 추가로 제작한 후 전체 코드를 출력하라

카테고리 페이지 수정

설명: 이전 제작한 내용을 다음과 같이 수정할 것이다. 수정될 내용은 다음 부분과 인덱스에 준수하여 제작하고 해당 프로젝트는 SPA방식으로 제작하는 것이 필수이며 PC와 모바일간의 UX를 고려해 제작해라

카테고리페이지
1. 콘텐츠영역은 width 1200px와 height 100vh에서 nav의 높이와 footer의 높이를 제외한 영역으로 구성된다.
2. 다음 작성되는 내용은 모두 콘텐츠 영역 내에 존재한다.

좌측 사이드바
1. 콘텐츠 영역 좌측에 존재하며 width는 10vw의 크기를 가진다.
2. 분류의 버튼 및 구성은 적당하나 가격은 progressbar형식에 좌우로 움직일 수 있는 버튼을 이용해 가격을 직접 계산하는 것을 추가해라.

메인영역
1. 상단에는 Home페이지에서 제작한 슬라이드와 동일한 swiper와 기능을 이용해 동작한다.
2. 현재 데이터가 없기 때문에 단색 카드를 이용해라
3. 하단에는 데이터에 저장된 내용을 grid형식으로 출력하라
4. 내용은 좌측 사이드바에서 클릭 및 분류를 통해 이에 해당하는 내용을 출력하라
5. 만일 100vh에서 nav의 높이와 메인의 상단 슬라이드의 높이  footer의 높이를 뺀 크기를 넘어가는 경우 메인 영역에서 상단의 슬라이드를 제외한 부분에 따로 스크롤을 적용해 해당 영역만 슬라이드 가능하도록 제작해라

더미데이터
1. 현재 8개가 존재한다 내부 값이나 데이터를 랜덤으로 작성하며 40개가 될때까지 더미데이터를 추가해라

위 내용 및 추가사항을 준수해 코드를 수정 및 제작하고 전체 내용을 출력해라

카테고리 페이지 수정

메인의 슬라이더의 문제점을 인식시킨 후 수정하고 메인 그리드의 출력 방식을 수정한다. 수정될 내용은 다음 부분에 정의되며 인덱스에 준수하여 수정하고 해당 페이지는 SPA방식으로 제작한다. 또한 PC와 모바일간의 UX를 고려해라.

메인 슬라이드
1. 현재 슬라이드 내부 기능은 유지한다.
2. next와 prev 버튼의 위치를 수정한다. 
3. 슬라이더 내부에 좌측 끝과 우측 끝에 위치하도록한다.
4. 예시로는 home페이지의 슬라이더 디자인을 참고하고 이처럼 버튼의 위치를 수정한다.

메인 그리드
1. 각 그리드의 셀 내부에 메이커도 표시되도록 수정한다.
2. 화면의 배율을 변결하더라도 nav와 메인, footer의 높이 합이 100vh 측 화면에 스크롤이 발생되지 않도록 수정하라.

위 내용을 인덱스에 준수하여 수정하되 수정이 필요하지 않은 부분은 유지하도록한다.
수정 후 수정된 부분을 설명하고 전체 내용을 출력하라.
