EVENT페이지 작성

이벤트 페이지를 만든다. home페이지에서 제작한 user사이드바, nav, footer는 동일한 width와 height로 사용한다. 해당 페이지는 3개의 큰 섹션을 가지며 현재 이는 동일한 내용으로 작성한다. 아래는 한 섹션에 대한 구조로 이와 동일하게 섹션 2개를 더 만들어라. 하단의 내용을 인덱스와 내용을 준수하여 제작하고 SPA방식으로 제작하는 것은 필수이다. PC와 모바일간의 UX를 고려하여 제작하라.

섹션의 구조는 상단, 중단, 하단으로 구성되며 비율은 3대 2대 5이며 width는 최대 1200px를 가진다.

상단
1. home페이지의 swiper를 사용한 슬라이더를 이용한다.
2. 총 3개의 배너를 가지며 next와 prev는 배너 내부에 존재한다.
3. 각 배너는 값이 없으므로 단색카드를 이용하며 몇번 배너인지는 배너의 정 중앙에 흰색글씨로 표시한다.
4. 3초 후 자동 슬라이드된다.
5. 상단에는 해당 swiper를 이용한 슬라이드만 존재하며 해당 영역을 모두 채운다.

중단
1. 한 줄에 분류버튼 5개가 존재한다.
2. 이를 이용해 하단의 grid에 출력되는 내용이 수정된다.
3. 총 3줄이 출력된다.

하단
1. 해당 영역 전체는 grid 영역이다.
2. grid에는 한 행에 6개씩 출력되며 중단에서 분류를 클릭하지 않은 경우 모든 아이템을 출력한다.
3. 중단에서 분류를 클릭하는 경우 바로바로 grid영역에는 포함되는 아이템을 출력한다.
4. 각 아아템에는 이미지(없으므로 단색 카드로 대체), 이름, 가격을 출력한다.
5. 섹션 기준으로는 모두 다른 것으로 보기 때문에 다른 섹션에는 영향을 미치지 않는다.
