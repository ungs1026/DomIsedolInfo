이벤트페이지 기능 수정

이벤트 페이지의 각 섹션에 존재하는 grid영역을 수정한다. 그 외 배너 및 분류기능의 3개의 행은 동일하게 유지된다. 다음 내용을 확인하고, 각 섹션의 요구사항을 준수하여 제작하라. SPA형태로 개발해야하며 PC와 모바일 간 UX를 고려하여 동일한 사용자 경험을 제공하라.

다음 내용은 페이지에 존재하는 3개의 섹션 모두 동일하게 적용된다. 또한 그리드 영역 외 슬라이드와 3행의 분류버튼들의 기능 및 디자인은 수정하지 않는다. 다음 선언하는 내용만 수정한다. 또한 다음 영역에 존재하는 기능을 수정하는 내용으로 대체한다.

섹션의 grid영역 수정
1. 그리드 영역에는 2행의 아이템만 출력되며 해당 영역의 크기도 이에 맞춘다.
2. 행의 2줄을 넘어가는 아이템은 출력되지 않는다.
3. 그리드영역 밑에 더보기 버튼이 존재하며 이는 해당 영역의 가운데에 존재한다.
4. 더보기 버튼을 누르는 경우 2행 이후 출력되지 않은 아이템이 출력되며 그리드 영역 높이가 증가한 아이템의 행 수만큼 증가한다.
5. 더보기 버튼을 누른경우 해당 버튼은 닫기로 수정된다.
6. 닫기 버튼을 누르는 경우 처음처럼 그리드 영역에 아이템 2행만큼만 표시되며 외의 아이템은 보이지 않고 버튼도 다시 더보기로 수정된다.

위 내용을 인덱스에 준수하여 수정하되 수정이 필요하지 않은 부분은 유지하도록한다. 수정 후 수정된 부분을 설명하고 전체 내용을 한 파일로 출력하라.

이벤트페이지 섹션의 그리드 영역 수정

다음 내용은 페이지에 존재하는 3개의 섹션 모두 동일하게 적용된다. 또한 그리드 영역 외 슬라이드와 3행의 분류버튼들의 기능 및 디자인은 수정하지 않는다. 다음 선언하는 내용만 수정한다. 또한 다음 영역에 존재하는 기능을 수정하는 내용으로 대체한다.

섹션의 grid영역 수정
1. 더보기 버튼을 누른 경우와 누르지 않은 경우로 분리되며 첫 시작은 더보기버튼을 누르지 않은 경우이다. 더보기 버튼을 누르지 않은 경우를 더보기, 더보기 버튼을 누른 경우 닫기로 명명한다.
1-1. 더보기
1-1-1. 세로로 그리드영역, 닫기 버튼 영역 2개의 영역이 있다. 이는 세로로 정렬되어 있으며 상위 div의 크기를 이어받는다.(width 100%)
1-1-2. 그리드 영역
1-1-2-1. 아이템의 행 2개의 높이를 가진다.
1-1-2-2. 영역을 초과하는 그리드는 보이지 않으며 이는 임시로 display none으로 설정한다. none으로 설정하는 이유는 해당 그리드 영역에서 스크롤이 발생하지 않기 위함이다.

1-1-3. 버튼 영역
1-1-3-1. 내부에 더보기 혹은 닫기로 작성된 버튼이 존재한다.
1-1-3-2. 이는 해당 버튼 영역 가운데 존재한다. 
1-1-3-3. 처음은 더보기로 이 버튼을 누르는 경우 닫기로 내부 값이 변한다.

1-2. 닫기
1-2-1. 1-1-2-2에서 영역을 초과한 아이템을 보여주며 초과한 아이템의 높이만큼 그리드영역의 높이가 증가한다.
1-2-2. 버튼이 눌리는 경우 1-1-2-1과 1-1-2-2의 기능으로 돌아간다.

위 내용을 인덱스에 준수하여 수정하되 수정이 필요하지 않은 부분은 절대로 수정하지 않는다.. 수정 후 수정된 부분을 설명하고 전체 내용을 한 파일로 출력하라.
