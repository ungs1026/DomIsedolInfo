가장 마지막으로 첨부했던 파일을 전체 코드로 생각하고 다음 요구사항에 맞춰 코드를 수정 및 추가한다.

1. 이를 vue3문법을 사용하여 v-if를 통해 페이지를 전환 할 것이다.
2. dom을 각 페이지별로 분할한 뒤 v-if방식으로 가져오고 초기화 시점을 nav를 통해 페이지를 전환한 시점으로 잡는다.
3. 초기화는 페이지 내 기능이 정상적으로 작동할 수 있도록 하는 것이다.
4. palybar는 페이지 전환 시 새로고침 되면 안된다.
5. 페이지 이동 시 기존 페이지의 기능 등을 모두 정상적으로 동작하기 위해 초기화 시점도 중요하다.
6. 현재 페이지에 작용되는 기능은 모두 동일하게 작동해야 한다,
7. spa방식으로 제작되어야 한다.

 위 인덱스를 준수하여 페이지를 만든다. 페이지는 spa방식으로 구현한다. 요구사항을 제작하는데 필요한 라이브러리는 cdn 방식으로 가져온다. 디자인은 요구사항을 충족하는 동시에 PC와 모바일간의 UX를 고려하여 구성한다. 수정 혹은 추가가 필요한 코드를 출력하고 어디에 추가해야하는지 출력한다. 오류가 나거나 수정이 필요한 부분은 추후 다시 요구할 것이다.

3. JS에서 DOMContentLoaded를 사용해서 페이지 전환을 하는 것이 아닌 초기화를 함수로 설정하여 페이지 전환시 DOMContentLoaded내의 동작을 가져오는 방식으로 한다.
4. 다음 요구사항에 vue3 문법으로 마이그래이션을 할 것이기 때문에 적합하게 수정한다.
